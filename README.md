# Trading Platform (Confidential Trading Demo)

This repository contains a single React component demo `confidential-trading-zama.jsx` — a visual dashboard that simulates confidential trading using FHE, ZK, and MPC concepts.

This is a UI-only demo. To run locally as a minimal Vite app:

1. Install dependencies:

   npm install

2. Start dev server:

   npm run start

3. Open http://localhost:5173 and import the component into a small React app or mount it directly.

Notes:
- The component is self-contained and intended as a demo/visual. It does not perform real encryption or connect to any blockchain.
- If you want, I can scaffold a minimal Vite + React app that mounts the component and adds TypeScript, ESLint, and a tiny test.
